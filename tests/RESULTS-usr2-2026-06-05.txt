Key implementation notes
--------------------------------------------------------------------------------
USR2 launch pattern: nginx is started from a subshell with a 0.5s sleep:
  ( nginx -p PREFIX -c PREFIX/nginx.conf & sleep 0.5 )
This ensures nginx records ngx_parent=subshell_PID before the subshell exits,
then gets reparented to init (getppid()==1). nginx's USR2 guard checks
"ignore if getppid()==ngx_parent", which evaluates to "1 == subshell_PID"
= false → USR2 is honoured.

With daemon off the exporter is a direct child of the master (SIGCHLD works
correctly for graceful-quit tracking).
